# 需求分析 · 南华期货自动下单（首版 MVP）

> 本文件为 TaskBundle 业务摘要；**机读验收规格**以 `期货策略-可验收回测规格.md` 为准。

## 目标

在南华期货场景下实现 Python 回测/模拟下单软件：计算技术指标 → 判定多空信号 → 模拟券商执行 → 输出信号序列与回测摘要。**首版不接实盘**。

## 核心能力

1. **指标**：K 线均线 5+6+7+8+9+11+20；BOLL 20+2；VOL 3+100；MACD 14+53+60；CCI 89
2. **空信号（3 分钟）**：前五线并拢后下穿 20 日线；非横盘；VOL 白线升 + 绿柱倍量；MACD 近零轴绿柱加长；CCI 二次下穿 0 轴；1 分钟同向下穿 20 日线；上证 + 深证均在 20 日线下
3. **多信号（3 分钟）**：镜像多头条件（详见规格 §4.2）
4. **止损 15 点**：区分昨日对冲单与当日开单规则（规格 §5）
5. **交付结构**：`config.yaml`、`indicators/`、`signals/`、`risk/`、`broker/`、`main.py`、pytest、`DELIVERY.md`

## 非目标

- 不证明策略盈利
- 不接实盘、不连券商 API
- 不做参数寻优 / ML
